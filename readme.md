# Calculadora de Enfestos

## Descrição

A Calculadora de Enfestos é uma ferramenta projetada para auxiliar na produção em massa de produtos da indústria têxtil. Seu principal propósito é calcular a distribuição eficiente de peças completas em uma folha de tecido, considerando a quantidade total de peças a serem cortadas e a metragem da folha de tecido disponível. 

## Funcionalidades

- Calcula a distribuição de peças completas em uma folha de tecido.
- Ajuda a otimizar o uso de material têxtil, reduzindo o desperdício.
- Interface intuitiva para facilitar a entrada de dados e visualização dos resultados.

## Tecnologias Utilizadas

- HTML e CSS para estrutura e interface do projeto.
- Javascript puro para a lógica da calculadora.
- Framework ElectronJS para o port do formato web em formato desktop.
